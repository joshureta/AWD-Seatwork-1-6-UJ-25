## Overview

<!-- TODO: To be changed -->
<!-- The following are just sample -->
The project is a Grade Management System designed to compute the General Weighted Average (GWA) for a student across 5 subjects. Its purpose is to simplify the grade computation process by automatically calculating the GWA, eliminating the need for manual formula application. This reduces the chances of error and ensures accurate results since users only need to input the grades, and the system will handle the calculations.
